# dotfiles

My Droid skills/scripts

## Scripts

| Script | Description |
|--------|-------------|
| **droid-mod** | Modify Droid CLI binary to disable truncation and unlock features |

```bash
python3 scripts/droid-mod/apply.py              # apply all mods
python3 scripts/droid-mod/apply.py 1,4,8        # apply specific mods
python3 scripts/droid-mod/apply.py status        # check status
python3 scripts/droid-mod/apply.py restore       # restore backup
```

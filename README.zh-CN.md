# dotfiles

我的 Droid skills/scripts

## Scripts

| 脚本 | 说明 |
|------|------|
| **droid-mod** | 修改 Droid CLI 二进制，禁用截断并解锁功能 |

```bash
python3 scripts/droid-mod/apply.py              # 应用全部 mod
python3 scripts/droid-mod/apply.py 1,4,8        # 应用指定 mod
python3 scripts/droid-mod/apply.py status        # 查看状态
python3 scripts/droid-mod/apply.py restore       # 恢复备份
```
